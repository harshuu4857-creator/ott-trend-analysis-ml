# 🎬 OTT Trend Analysis (Machine Learning Project)

## 📌 Overview
This project analyzes OTT platform trends using the Netflix dataset and builds a Machine Learning model to classify content as **Movie or TV Show**.

It is an end-to-end data analytics + machine learning project with data cleaning, visualization, and deployment using Streamlit.

---

## 🚀 Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning Model (Random Forest)
- Streamlit Web Application

---

## 📊 Key Insights
- Movies are more in number compared to TV Shows
- Content production increased rapidly after 2015
- USA produces the highest amount of content
- Most content falls under the "TV-MA" rating category
- Duration plays an important role in classification

---

## 🤖 Machine Learning Model
- **Algorithm Used:** Random Forest Classifier
- **Task:** Classification (Movie vs TV Show)
- **Input Features:**
  - Release Year
  - Duration
  - Rating
