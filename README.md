# 🎬 OTT Trend Analysis (Machine Learning Project)

## 📌 Overview

This project focuses on analyzing OTT platform trends using the Netflix dataset and building a Machine Learning model to classify content as **Movie or TV Show**.

It is an end-to-end project that includes data cleaning, exploratory data analysis (EDA), feature engineering, machine learning, and deployment using Streamlit.

---

## 🚀 Features

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Machine Learning Model (Random Forest)
* Interactive Streamlit Web App

---

## 📊 Key Insights

* Movies are more in number compared to TV Shows
* Content production increased significantly after 2015
* USA produces the highest amount of content
* Most content falls under "TV-MA" rating
* Duration is an important factor in classification

---

## 🤖 Machine Learning Model

* **Algorithm:** Random Forest Classifier
* **Task:** Classification (Movie vs TV Show)
* **Features Used:**

  * Release Year
  * Duration
  * Rating
